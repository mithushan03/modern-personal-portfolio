# Mithushan Portfolio

Modern personal portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run Locally

```bash
npm install
npm run dev
```

## Edit Content

Portfolio content lives in `src/data/portfolio.js`.

The intro background video is `public/assets/intro-video.mp4`, and the hero poster image is `public/assets/hero-poster.jpg`.
